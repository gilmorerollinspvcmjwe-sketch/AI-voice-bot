## 一、测试概述

### 1.1 测试目标

本文档旨在全面测试机器人的模拟对话调试功能，主要包括：

1. **模拟对话调试** (BotDebugConfig)：测试文本/语音对话模拟能力
2. **变量管理**：测试输入变量、对话变量、提取变量的监控与修改
3. **标签管理**：测试手动标签与自动打标功能
4. **延迟监控**：测试语音模式下的性能指标显示

### 1.2 调试功能模块清单

| 模块名称 | 文件路径 | 功能描述 |
|---------|---------|---------|
| BotDebugConfig | components/bot/BotDebugConfig.tsx | 模拟对话调试界面 |

## 二、测试环境

### 2.1 硬件环境

- **处理器**：Intel Core i5 或同等性能
- **内存**：8GB RAM
- **硬盘**：50GB 可用空间
- **显示器**：1920x1080 分辨率

### 2.2 软件环境

- **操作系统**：Windows 10/11 或 macOS
- **浏览器**：Chrome 90+ / Firefox 90+ / Edge 90+
- **Node.js**：v16+
- **开发框架**：React 18+
- **构建工具**：Vite 4+

### 2.3 测试数据准备

- 测试用机器人配置（BotConfiguration）
- 预设变量数据（inputVars、conversationVars、extractionVars）
- 标签组配置（LabelGroup）

## 六、附录

### 6.1 关键数据结构

```typescript
// 聊天消息
interface ChatMessage {
  id: string;
  role: 'user' | 'model' | 'system';
  content: string;
  timestamp: number;
  latency?: LatencyMetrics;
  tokens?: TokenMetrics;
  tags?: string[];
  actionEvent?: {
    type: 'TRANSFER' | 'HANGUP';
    detail: string;
    reason?: string;
  };
}

// 延迟指标
interface LatencyMetrics {
  asr?: number;
  llmTTFB?: number;
  llmStreamOutput?: number;
  llmTotal: number;
  ttsFirstAudio?: number;
  ttsFullPlayback?: number;
  ttsTotal?: number;
  e2eTTFB?: number;
  e2eTotal?: number;
}
```

### 6.2 模拟对话功能模块

```
BotDebugConfig (模拟对话调试)
    │
    ├── 聊天区域 (消息显示、输入)
    ├── 变量监视器
    │   ├── 输入变量 (可编辑)
    │   ├── 对话变量 (只读)
    │   └── 提取变量 (只读)
    ├── 标签管理
    │   ├── 系统标签 (自动打标)
    │   └── 手动标签 (用户选择)
    └── 语音控制面板
        ├── 录音模拟
        ├── 音色选择
        └── 延迟详情
```
