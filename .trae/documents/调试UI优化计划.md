# 调试UI优化计划

## 优化目标

参照 MicroFlowEditor 画布的 UI 风格，重新调整 IntentFlowDebugger 调试界面的视觉设计，使其更加美观、专业、一致。

## 当前问题分析

1. **颜色风格不一致**
   - 调试器使用深色主题（slate-900），而编辑器使用浅色主题（slate-50）
   - 需要统一或明确区分两种模式

2. **节点样式差异大**
   - 调试器节点：简单的矩形框，蓝色发光边框
   - 编辑器节点：圆角卡片，彩色分类，带图标

3. **布局不够专业**
   - 调试控制栏按钮排列不够紧凑
   - 右侧面板标签设计简单

4. **缺少视觉层次**
   - 执行路径动画不够明显
   - 节点状态标识不够清晰

## 优化方案

### 1. 整体布局风格统一

**参考 MicroFlowEditor：**
- 使用浅色背景（bg-slate-50）
- 卡片式布局，带阴影和圆角
- 边框使用 slate-200

**调试器调整：**
- 主画布：bg-slate-50
- 头部：bg-white border-b border-slate-200
- 右侧面板：bg-white border-l border-slate-200

### 2. 节点样式优化

**参考编辑器节点样式：**
```
- 圆角：rounded-lg
- 尺寸：w-[180px] h-[60px]
- 边框：border-2
- 图标：左侧小图标
- 文字：标题 + 子类型
```

**调试器节点增强：**
- 保持编辑器节点的颜色分类（ emerald/indigo/orange/amber/blue/rose/purple ）
- 执行状态通过边框颜色变化表示：
  - 等待中：默认边框
  - 执行中：ring-4 ring-blue-400 animate-pulse
  - 已完成：border-green-400
  - 错误：border-red-400
  - 跳过：border-gray-300 opacity-50
- 断点标记：右上角红色圆点
- 执行状态图标：右下角小图标

### 3. 调试控制栏优化

**参考编辑器工具栏：**
- 紧凑排列
- 图标 + 文字（小字号）
- 悬停效果

**控制栏设计：**
```
┌─────────────────────────────────────────────────────────┐
│ [▶ 开始] [⏸ 暂停] [⏭ 步过] [⏎ 步入] [⏹ 停止] [↺ 重置] │
│                    速度: [━━━●━━] 1.5x                   │
└─────────────────────────────────────────────────────────┘
```

### 4. 连线样式优化

**执行路径可视化：**
- 已执行路径：实线 + 渐变色（蓝色）
- 当前路径：虚线流动动画
- 未执行路径：灰色虚线

### 5. 右侧面板优化

**参考编辑器右侧面板：**
- 标签页设计：底部边框指示器
- 卡片式内容区域
- 紧凑的间距

**面板结构：**
```
┌─────────────────┐
│ [执行历史] [变量] │  ← 标签页
├─────────────────┤
│                 │
│   内容区域       │  ← 卡片式布局
│                 │
├─────────────────┤
│ 断点列表         │  ← 底部固定
└─────────────────┘
```

### 6. 颜色系统统一

**使用编辑器相同的颜色映射：**
| 节点类型 | 背景色 | 边框色 | 文字色 |
|---------|--------|--------|--------|
| START | emerald-50 | emerald-200 | emerald-700 |
| LISTEN | indigo-50 | indigo-200 | indigo-700 |
| BRANCH | orange-50 | orange-200 | orange-700 |
| AI_AGENT | amber-50 | amber-200 | amber-700 |
| LOGIC | blue-50 | blue-200 | blue-700 |
| ACTION | rose-50 | rose-200 | rose-700 |
| DATA | purple-50 | purple-200 | purple-700 |

### 7. 动画效果优化

**执行中动画：**
- 脉冲效果：animate-pulse
- 流动效果：SVG stroke-dasharray 动画
- 过渡动画：transition-all duration-300
