# 表格组件规范

## 1. 表格类型

| 类型 | 场景 | 默认密度 |
|---|---|---|
| 配置列表表格 | 机器人配置、工具配置、知识库列表 | regular，行高 64px |
| 日志表格 | 接口调用、AI 回复日志、通话记录 | compact，行高 52px |
| 选择表格 | 选择知识库、选择工具、选择队列 | regular，支持单选/多选 |
| 明细表格 | 工具返回、变量映射、参数列表 | compact，支持复制和截断 |
| 可编辑表格 | Header 参数、KeyValue、Schema 字段 | compact，单元格内编辑 |

## 2. 表格结构

| 区域 | 规格 |
|---|---|
| Toolbar | 高 56px，包含搜索、筛选、批量操作、新增按钮 |
| Header | 高 44px，浅灰底，字段名 12px/700 |
| Body Row | regular 64px，compact 52px |
| Cell | 左右 padding 14px，顶部底部 14px |
| Action Cell | 固定在右侧，宽度按操作数量定义 |
| Pagination | 高 48px，右对齐，显示总数和每页数量 |
| Bulk Bar | 选中后出现，高 48px，展示已选数量和批量动作 |

## 3. 列规范

| 列类型 | 展示规则 |
|---|---|
| 主标题列 | 加粗主标题 + 一行辅助描述；描述可截断 |
| 状态列 | 使用 Status Pill，不只靠颜色表达 |
| 时间列 | `YYYY-MM-DD HH:mm:ss`，窄屏可换为两行 |
| 数值列 | 右对齐；金额、耗时、次数单位明确 |
| 标签列 | 最多展示 2 个，更多显示 +N |
| 开关列 | 居中；变更后有保存反馈 |
| 操作列 | 最多 3 个直接操作；危险操作红色；更多进菜单 |
| 长文本列 | 单行截断；hover/点击查看完整内容 |
| 代码/JSON 列 | 等宽字体；默认摘要，详情弹窗查看 |

## 4. 搜索、筛选、排序

| 能力 | 规则 |
|---|---|
| 搜索 | 输入不立即清空列表；Enter 或防抖触发；清空恢复默认 |
| 筛选 | 筛选项超过 3 个时放入筛选面板；已选条件可见 |
| 排序 | 仅可排序列显示排序图标；支持升序/降序/取消 |
| 组合筛选 | 条件摘要显示在 toolbar 下方或筛选按钮旁 |
| 保存视图 | 高级场景可保存筛选组合；默认不展示 |

## 5. 选择与批量操作

| 场景 | 规则 |
|---|---|
| 单选 | 点击行或 radio 选中；再次点击不取消 |
| 多选 | checkbox 选择；表头支持全选/半选 |
| 跨页选择 | 必须明确提示“仅本页”或“全部结果” |
| 批量操作 | 只在选中后出现；危险批量操作二次确认 |
| 禁选行 | checkbox disabled，并说明原因 |

## 6. 固定、滚动与列宽

| 能力 | 规则 |
|---|---|
| 固定表头 | 长列表默认固定表头 |
| 固定列 | 主标题列可左固定；操作列可右固定；固定列有阴影分隔 |
| 横向滚动 | 表格宽度超过容器时启用；操作列保持可见 |
| 列宽调整 | 高级表格支持拖拽列宽；最小宽度不低于 120px |
| 虚拟滚动 | 超过 1000 行使用；hover、选中、固定列仍需稳定 |

## 7. 可编辑单元格

| 状态 | 规则 |
|---|---|
| readonly | 文本展示，可复制 |
| editing | 单元格内控件，不改变行高 |
| saving | 单元格 loading，阻止重复提交 |
| saved | 短暂成功反馈 |
| error | 单元格红框，错误文案可用 Tooltip 展示 |
| dirty | 未保存标记，离开页面需提示 |

## 8. 空、加载、错误与权限

| 状态 | 展示 |
|---|---|
| loading | 表头保留，body 使用 skeleton row |
| empty-data | 无数据：简短说明 + 可选新增按钮 |
| empty-filter | 筛选无结果：提示清空筛选 |
| load-error | 加载失败：错误原因 + 重试按钮 |
| no-access | 无权限：说明权限缺失，不展示假数据 |
| partial-error | 部分列失败：失败列展示错误标识，其他数据保留 |

## 9. 常见表格列模式

| 表格 | 推荐列 |
|---|---|
| 配置列表 | 名称、描述、模型/类型、最后更新时间、状态、操作 |
| 字段配置列表 | 字段名、数据类型、枚举/规则、输出状态、最近命中、操作 |
| 外部能力列表 | 名称、类型、鉴权方式、返回处理、最近调用、状态 |
| 事件日志列表 | 轮次、触发类型、位置、模型、首响、总耗时、状态 |
| 召回结果列表 | 来源、文档、标题、分数、重排分、片段摘要 |
