# 表单组件规范

## 1. 表单布局

| 类型 | 用法 | 规格 |
|---|---|---|
| 单列表单 | 配置项较少、文本编辑、弹窗 | label 在上，字段宽度 100%，行距 16px |
| 双列表单 | 基础配置、模型参数、TTS 参数 | 两列等宽，列间距 24px，移动端降为单列 |
| 标签左置表单 | 高密度后台参数 | label 宽 160px，控件区自适应，行高不低于 40px |
| 分组表单 | 多类配置同屏 | 使用 section/card 分组，组间距 20px |
| 内联表单 | 筛选、搜索、工具栏 | 控件高度一致，间距 10px，按钮右对齐 |

## 2. 基础输入组件

| 组件 | 场景 | 尺寸 | 交互与校验 |
|---|---|---|---|
| Input | 名称、手机号、ID、短文本 | 高 40px，圆角 10px，左右 padding 12px | 支持 clear；错误在下方；不把规则写进 placeholder |
| SearchInput | 列表搜索、知识搜索、变量搜索 | 高 38px，圆角 999px | Enter 搜索；清空恢复；搜索中保留旧结果 |
| NumberInput | 温度、阈值、次数、金额 | 高 40px，右侧可显示单位 | 限制 min/max/step；非法输入保留并提示 |
| Textarea | 描述、话术、说明 | 最小高 96px | 支持滚动和 resize；长度超限显示计数 |
| TextEditor | 长文本、Prompt、规则说明 | 最小高 360px，长文本滚动 | 变量高亮；支持插入变量；保存前校验未闭合变量 |

## 3. 选择组件

| 组件 | 场景 | 行为 |
|---|---|---|
| Select | 模型、语言、状态、队列 | 选项少于 8 条；菜单宽度不小于触发器 |
| Searchable Select | 知识库、工具、流程节点 | 支持关键词搜索；加载态不清空已选值 |
| MultiSelect | 标签、可用工具、知识库范围 | tag 超出折叠为 +N；支持清空全部 |
| Combobox | 可选也可输入的业务枚举 | 明确区分“选择已有”和“创建新值” |
| Radio Group | 互斥策略，如打断策略 | 选项不超过 5 个；必须展示当前选项含义 |
| Checkbox Group | 多个独立开关，如日志字段 | 支持全选/半选；组标题不可省略 |
| Switch | 启用/停用、允许打断、Rerank | 不承载复杂选择；重要开关保存后反馈结果 |
| Slider | 温度、音量、语速、阈值 | 必须同步数字值；展示推荐范围 |

## 4. 时间、文件、结构化编辑

| 组件 | 场景 | 规格 |
|---|---|---|
| DatePicker | 外呼计划、报表时间 | 禁用不可选日期；错误说明原因 |
| TimePicker | 提醒时间、重拨时间 | 支持 24 小时制；可配置分钟步长 |
| DateRangePicker | 报表筛选、日志查询 | 开始结束同屏；快捷选项不超过 6 个 |
| FileUpload | 知识导入、附件上传 | 拖拽态、上传进度、失败重试、格式/大小提示 |
| KeyValueEditor | Header、URL 参数、变量映射 | 行高 40px；支持增删排序；空 key 禁止保存 |
| JSONEditor | 工具返回清洗、接口样例 | 等宽字体；错误定位到行列；支持格式化 |
| SchemaEditor | 工具入参/出参结构 | 字段名、类型、是否必填、说明、默认值 |

## 5. 高级编辑组件

| 组件 | 用法 | 规则 |
|---|---|---|
| Variable Picker | 在长文本、话术、接口参数中插入变量 | 插入格式统一为 `{{变量名}}`；变量缺失时红色标记 |
| Rule Editor | 配置字段抽取、分类、判断等规则 | 字段名、含义、枚举、类型、状态输出必须分离 |
| Response Mapper | 配置接口返回清洗摘要 | 原始返回、处理结果、给模型使用的摘要分区展示 |
| Interruption Rule Editor | 配置是否允许打断 | 区分可打断、禁打断、固定播报等策略 |
| Trigger Editor | 配置开始、结束、定时等触发器 | 条件、动作、失败处理分区展示 |
| Segment Viewer | 展示多来源长文本拼接 | 默认展示全量；展开查看拼接来源 |

## 6. 状态规格

| 状态 | 视觉 | 规则 |
|---|---|---|
| default | 白底默认边框 | 可输入 |
| hover | 边框略深 | 不改变尺寸 |
| focus | 蓝色 focus ring | 键盘可见 |
| filled | 正常文字 | 不覆盖用户输入 |
| readonly | 浅灰底正常文字 | 可复制不可编辑 |
| disabled | 降透明度 | 不触发事件 |
| loading | spinner 或骨架 | 保留已选值 |
| error | 红边框+错误文案 | 错误文案在控件下方 |
| success | 成功提示或绿色状态 | 不长期占用空间 |
| dirty | 未保存标记 | 离开前提示 |

## 7. 文案与溢出

- Label 使用业务名词，不使用技术字段名作为唯一说明。
- Placeholder 只提示输入样例，不承载规则。
- 帮助文案 1 行为宜；复杂说明用 Tooltip/Popover。
- 错误文案说明原因和修正方式，如“手机号需为 11 位数字”。
- 长字段值单行截断，hover 查看完整值；Prompt/JSON 类字段内部滚动。
